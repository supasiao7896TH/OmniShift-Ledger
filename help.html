การตั้งค่าเบื้องต้น

02

ก่อนเริ่มบันทึกข้อมูล ควรตั้งค่าข้อมูลส่วนตัวและระบบกะให้ครบถ้วน

💵 เงินเดือนและอัตราค่าตอบแทน

ไปที่แท็บ Settings แล้วกรอกข้อมูลต่อไปนี้

รายการค่าเริ่มต้นคำอธิบาย
เงินเดือน—เงินเดือนฐาน (บาท/เดือน)
ค่ากะเช้า150 บาท/กะเบี้ยเลี้ยงกะเช้า
ค่ากะดึก200 บาท/กะเบี้ยเลี้ยงกะดึก
ชั่วโมงทำงาน/เดือน176 ชม.ใช้คำนวณอัตรา OT ต่อชั่วโมง
เบี้ยที่พัก—คิดเป็น % จากเงินเดือน
โบนัสวันหยุด—คิดเป็น % จากเงินเดือน

🔄 ตั้งค่าการหมุนเวียนกะ — ส่วนสำคัญที่สุด

ระบบใช้รูปแบบ 4 ทีม (A/B/C/D) วนรอบ 12 วัน โดยแต่ละทีมจะหมุนเวียนกันระหว่างกะเช้า กะดึก และวันหยุด

ทีม A ทีม B ทีม C ทีม D

วิธีตั้งค่า: ไป Settings → ตั้งค่าการหมุนเวียนกะ

  1. 1กด "เพิ่ม segment"
  2. 2เลือก "วันที่เริ่มต้น" — วันแรกที่คุณรู้ว่าตัวเองอยู่ทีมไหน
  3. 3เลือก "ทีม" ของตัวเอง (A / B / C / D)
  4. 4กด "บันทึกและเติมกะ" — ระบบจะคำนวณกะของทุกวันอัตโนมัติ
⚠️

หากถูกย้ายทีม: กด "เพิ่ม segment" อีกอัน แล้วระบุ วันที่เริ่มทีมใหม่ และ ทีมใหม่ ระบบจะปรับกะนับจากวันนั้นโดยอัตโนมัติ

หลังตั้งค่าแล้ว ปฏิทินจะแสดงสัญลักษณ์บนแต่ละวัน เช่น ☀A ●D หรือ ☀C ●B

  • ☀ = ทีมที่ทำกะเช้าวันนั้น
  • ● = ทีมที่ทำกะดึกวันนั้น

การบันทึกกะรายวัน

03
  1. 1กดที่วันในปฏิทินที่ต้องการบันทึก
  2. 2หน้าแก้ไขวันจะเปิดขึ้น เลือกประเภทกะ
  3. 3บันทึก OT (ถ้ามี) แล้วกด "บันทึก"

ประเภทกะ

☀️ เช้า (M) 🌙 ดึก (N) 🏠 หยุด

การบันทึก OT

ใส่จำนวนชั่วโมง OT แยกตามอัตรา

1.5× OT วันธรรมดา — ทำเกินเวลาปกติในวันทำงาน
2.0× OT วันหยุดสัปดาห์ — ทำงานในวันที่ควรหยุด
3.0× OT วันหยุดนักขัตฤกษ์ — วันหยุดราชการ / วันหยุดพิเศษ
🔵

วันที่มีการบันทึก OT จะแสดง จุดสีส้ม ที่มุมบนซ้ายของช่องวันในปฏิทิน ทำให้มองเห็นได้ง่าย

🏖️

การบันทึกการลา

04
  1. 1กดที่วันที่ต้องการบันทึกการลาในปฏิทิน
  2. 2เลือกกะเป็น "หยุด"
  3. 3เลือกเหตุผลการลา
  4. 4กรอกรายละเอียดเพิ่มเติม (ถ้ามี) แล้วกด "บันทึก"

ประเภทการลา

  • ลาป่วย — ไม่สบาย มีใบรับรองแพทย์หรือไม่ก็ได้
  • ลากิจ — ธุระส่วนตัว
  • ลาพักร้อน — วันลาพักผ่อนประจำปี
  • ลาอื่นๆ — การลาประเภทอื่น เช่น ลาคลอด ลาบวช

บันทึกผู้แทน

ถ้ามีคนมาทำงานแทนในวันที่ลา ให้ใส่ชื่อผู้แทนในช่อง "รายละเอียด"

📊

ระบบจะนับสถิติการลาและคนที่มาแทนอัตโนมัติ สามารถดูสรุปได้ในแท็บ "สถิติ"

🔄

การแทนกะ

05

ระบบรองรับการบันทึกทั้ง 2 กรณี

📤 ฉันไปแทนคนอื่น

  • กดวันที่ไปแทน → เลือกกะ (เช้า/ดึก)
  • เลือกเหตุผลเป็น "แทนกะ"
  • ใส่ ชื่อคนที่ตัวเองไปแทน ในช่อง "รายละเอียด"

📥 มีคนมาแทนฉัน

  • กดวันที่หยุด → เลือกกะเป็น "หยุด"
  • เลือกเหตุผลเป็นประเภทการลาที่เหมาะสม
  • ใส่ ชื่อผู้ที่มาแทน ในช่อง "รายละเอียด"
📈

ดูสถิติการแทนกะทั้งหมด รวมถึงรายชื่อคนที่แทนและถูกแทนได้ในแท็บ "สถิติ"

💰

สรุปรายได้

06
  1. 1เลื่อนลงใต้ปฏิทิน กด "This Month's Summary"
  2. 2หน้าสรุปรายได้ประจำเดือนจะเปิดขึ้น

รายการที่แสดงในสรุป

  • เงินเดือนฐาน — ตามที่ตั้งค่าไว้
  • ค่ากะ — รวมค่ากะเช้า + ค่ากะดึก ทุกกะในเดือนนั้น
  • OT รวม — แยกแสดง 1.5× / 2.0× / 3.0× พร้อมจำนวนชั่วโมงและเงิน
  • เบี้ยเลี้ยง — ตามที่ตั้งค่าเป็น %
  • โบนัส — ตามที่ตั้งค่าเป็น %
  • รายได้รวม — ผลรวมทั้งหมด

กราฟรายได้

  • Income Breakdown — กราฟวงกลม (Donut) แสดงสัดส่วนรายได้แต่ละประเภท
  • Monthly OT Hours — กราฟแท่งแสดงชั่วโมง OT แต่ละเดือน

AI วิเคราะห์ (Gemini AI)

07
🔑

ฟีเจอร์นี้ต้องใช้ Gemini API Key ซึ่งสมัครฟรีได้ที่ aistudio.google.com

วิธีตั้งค่า API Key

  1. 1ไปที่ Settings → ตั้งค่า Gemini AI
  2. 2วาง API Key ที่ได้รับในช่องที่กำหนด
  3. 3กด "บันทึก"

วิธีใช้งาน

  1. 1เปิดหน้า This Month's Summary
  2. 2กดปุ่ม "🤖 AI วิเคราะห์กะ & รายได้"
  3. 3รอสักครู่ ผลวิเคราะห์จะแสดงขึ้น

AI จะวิเคราะห์อะไรบ้าง?

  • ภาพรวมเดือนนี้ — สรุปการทำงาน จำนวนกะ OT และรายได้รวม
  • แนวโน้มรายได้ — เปรียบเทียบกับเดือนก่อนหน้า
  • แนะนำ OT เดือนหน้า — คำแนะนำเพื่อเพิ่มรายได้หรือลด Burnout

สำรอง / กู้คืนข้อมูล

09

📤 สำรองข้อมูล (Backup)

  1. 1ไปที่ Settings → สำรองข้อมูล (Backup)
  2. 2กดปุ่ม "ดาวน์โหลด"
  3. 3ไฟล์ .json จะถูกบันทึกลงเครื่อง

📥 กู้คืนข้อมูล (Restore)

  1. 1ไปที่ Settings → กู้คืนข้อมูล (Restore)
  2. 2กด "เลือกไฟล์" แล้วเลือกไฟล์ .json ที่สำรองไว้
  3. 3กด "กู้คืน" เพื่อยืนยัน
⚠️

คำแนะนำ: สำรองข้อมูลอย่างน้อยเดือนละครั้ง หรือก่อนเปลี่ยนโทรศัพท์ / ล้างเครื่อง เพื่อป้องกันข้อมูลสูญหาย

🔒

ความเป็นส่วนตัว

10
  • ข้อมูลทั้งหมดเก็บใน localStorage ของ Browser บนเครื่องผู้ใช้เท่านั้น
  • ไม่มี server กลาง — ไม่มีการส่งข้อมูลออกสู่ภายนอก
  • แต่ละคนที่ติดตั้งแอปจะมีข้อมูล แยกกัน 100% ไม่มีการเชื่อมโยงกัน
  • ข้อมูลจะยังอยู่แม้ปิดแอปหรือรีสตาร์ทเครื่อง
🤖

ข้อยกเว้น Gemini AI: เมื่อกดปุ่ม "AI วิเคราะห์" ข้อมูลสรุปการทำงานจะถูกส่งไปยัง Google Gemini API เพื่อวิเคราะห์ ซึ่งผู้ใช้เป็นผู้กดเองโดยสมัครใจ และใช้ API Key ของตัวเอง

💡

หากล้างข้อมูลเบราว์เซอร์ (Clear Cache / Clear Data) ข้อมูลแอปจะหายทั้งหมด ควรสำรองข้อมูลไว้เป็นประจำ
