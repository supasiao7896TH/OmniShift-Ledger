🔄 ตั้งค่าการหมุนเวียนกะ — ส่วนสำคัญที่สุด

ระบบรองรับ 4 ทีม (A/B/C/D) และให้ เลือกรูปแบบกะได้ 2 แบบ โดยแต่ละทีมหมุนเวียนกันระหว่างกะเช้า กะดึก และวันหยุด

🆕 เลือกรูปแบบกะ (Shift Pattern)

ไป Settings → การจัดการ → รูปแบบกะ แล้วเลือกจาก dropdown:

  • รอบ 12 วัน (3 ทำ 3 หยุด) — ทำงาน 3 วัน / หยุด 3 วัน สลับเช้า–ดึก (รูปแบบเดิม)
  • รอบ 14 วัน (GC: 3-2 / 2-3 / 2-2) — ทำงาน 3-2 วัน / หยุด 2-3 วัน สลับเช้า–ดึกทุก loop รอบเต็ม 28 วัน (ทำงาน 50%) อ้างอิงตาราง GC
ℹ️

เปลี่ยนรูปแบบกะแล้ว ข้อมูลที่ลงไว้ไม่ถูกลบ — ต้องเปิด "ตั้งค่าการหมุนเวียนกะ" แล้วกด "นำไปใส่ในปฏิทิน" อีกครั้ง เพื่อให้ปฏิทินอัปเดตตามรูปแบบใหม่

ทีม A ทีม B ทีม C ทีม D

วิธีตั้งค่า: ไป Settings → ตั้งค่าการหมุนเวียนกะ

  1. 1ตรวจ "วันอ้างอิง" ด้านบน — แบบ GC ระบบตั้งให้อัตโนมัติ (14 ม.ค. 2026) ไม่ต้องแก้ · แบบ 12 วัน ให้กรอกวันที่ทีม A เข้ากะเช้าวันแรกจากตารางโรงงาน
  2. 2กด "+ เพิ่มช่วง" → เลือก วันที่เริ่ม และ ทีม ของตัวเอง (A / B / C / D)
  3. 3เลือก จำนวนเดือนที่ต้องการเติม
  4. 4กด "นำไปใส่ในปฏิทิน" — ระบบจะคำนวณกะของทุกวันอัตโนมัติ (OT / เหตุผล ที่ลงไว้ไม่ถูกลบ)
⚠️

หากถูกย้ายทีม: กด "+ เพิ่มช่วง" อีกอัน แล้วระบุ วันที่เริ่มทีมใหม่ และ ทีมใหม่ ระบบจะปรับกะนับจากวันนั้นโดยอัตโนมัติ

หลังตั้งค่าแล้ว ปฏิทินจะแสดงสัญลักษณ์บนแต่ละวัน เช่น 🌞A 🌙D หรือ 🌞C 🌙D

  • 🌞 = ทีมที่ทำกะเช้า (M)วันนั้น
  • 🌙 = ทีมที่ทำกะดึก (N)วันนั้น

03
  1. 1กดที่วันในปฏิทินที่ต้องการบันทึก
  2. 2หน้าแก้ไขวันจะเปิดขึ้น เลือกประเภทกะ
  3. 3บันทึก OT (ถ้ามี) แล้วกด "บันทึก"

Badgeชื่อกะเวลาชม.ค่ากะOT

ℹ️

กะเดย์ (D) คือการทำงานวันจันทร์–ศุกร์ 8 ชั่วโมง ไม่มีค่าเบี้ยเลี้ยงกะ แต่ยังได้รับเงินเดือน เบี้ยที่พัก และสิทธิ์ OT 1.5× เหมือนกะปกติ

การบันทึก OT

ใส่จำนวนชั่วโมง OT แยกตามอัตรา

1.5× OT วันธรรมดา / กะเดย์ — ทำเกินเวลาปกติในวันทำงาน
2.0× OT วันหยุดสัปดาห์ — ทำงานในวันที่ควรหยุด
3.0× OT วันหยุดนักขัตฤกษ์ — วันหยุดราชการ / วันหยุดพิเศษ
🔵

วันที่มีการบันทึก OT จะแสดง จุดสีส้ม ที่มุมบนขวาของช่องวันในปฏิทิน ทำให้มองเห็นได้ง่าย

⚠️ คำเตือนกะต่อเนื่อง (N → M / D)

หากบันทึกกะเช้า (M) หรือกะเดย์ (D) ต่อจากกะดึก (N) ของวันก่อนหน้า ระบบจะแสดงคำเตือนว่าทำงานต่อเนื่องมาแล้ว 12 ชั่วโมง เพื่อให้ยืนยันก่อนบันทึก

การแทนกะ

05

ระบบรองรับการบันทึกทั้ง 2 กรณี

สรุปรายได้

06
  1. 1เลื่อนลงใต้ปฏิทิน กด "This Month's Summary"
  2. 2หน้าสรุปรายได้ประจำเดือนจะเปิดขึ้น

รายการที่แสดงในสรุป

  • เงินเดือนฐาน — ตามที่ตั้งค่าไว้
  • ค่ากะ — รวมค่ากะเช้า (M) + ค่ากะดึก (N) — กะเดย์ (D) ไม่มีค่ากะ
  • OT รวม — แยกแสดง 1.5× / 2.0× / 3.0× พร้อมจำนวนชั่วโมงและเงิน
  • เบี้ยที่พัก — คำนวณจาก % เงินเดือน (ขั้นต่ำ 8,000 บาท)
  • โบนัส — ตามที่ตั้งค่าเป็น %
  • รายได้รวม — ผลรวมทั้งหมด

แถบสรุปด่วน (Summary Bar)

ด้านล่างปฏิทินมีแถบ 4 ช่องแสดงตัวเลขด่วนของเดือนปัจจุบัน

สำรอง / กู้คืนข้อมูล

09

📤 สำรองข้อมูล (Backup)

  1. 1ไปที่ Settings → สำรองข้อมูล (Backup)
  2. 2กดปุ่ม "ดาวน์โหลด"
  3. 3ไฟล์ .json จะถูกบันทึกลงเครื่อง

📥 กู้คืนข้อมูล (Restore)

  1. 1ไปที่ Settings → กู้คืนข้อมูล (Restore)
  2. 2กด "เลือกไฟล์" แล้วเลือกไฟล์ .json ที่สำรองไว้
  3. 3กด "กู้คืน" เพื่อยืนยัน
⚠️

คำแนะนำ: สำรองข้อมูลอย่างน้อยเดือนละครั้ง หรือก่อนเปลี่ยนโทรศัพท์ / ล้างเครื่อง เพื่อป้องกันข้อมูลสูญหาย

วันหยุดพิเศษส่วนตัว

10

นอกจากวันหยุดราชการที่ระบบบันทึกไว้แล้ว คุณสามารถเพิ่มวันหยุดเฉพาะของโรงงาน หรือวันหยุดพิเศษอื่นๆ ได้เอง

วิธีเพิ่มวันหยุดส่วนตัว

  1. 1ไปที่ Settings → วันหยุดส่วนตัว
  2. 2เลือก "วันที่เริ่ม"
  3. 3เลือก "วันที่สิ้นสุด" (ถ้าเป็นวันเดียว ปล่อยว่างได้)
  4. 4ใส่ ชื่อวันหยุด แล้วกด "เพิ่ม"
📅

รองรับทั้ง วันเดียว (เลือกแค่วันเริ่ม) และ หลายวันต่อเนื่อง (เลือกทั้งวันเริ่มและวันสิ้นสุด) เช่น หยุดช่วงเทศกาล 3 วัน

การแสดงผลในปฏิทิน

  • วันหยุดส่วนตัวจะแสดงพื้นหลัง สีม่วงอ่อน ต่างจากวันหยุดราชการ (สีแดงอ่อน)
  • ชื่อวันหยุดจะแสดงในช่องวันด้วย ตัวอักษรสีม่วง
  • หากชื่อยาวเกินไปจะถูกตัดด้วย … เพื่อไม่ให้ layout เสีย
